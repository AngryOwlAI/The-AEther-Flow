# Director Decision Record DDR-20260611-001

## Current Objective

Create and execute one bounded `validator-engineer@0.1.0` task overlay for
Mermaid documentation governance. The job adds a Mermaid Documentation subskill
under `visual-explainer`, a narrow validator for registered explanatory
Markdown and tracked HTML Mermaid parity, local runtime asset policy, bootstrap
integration, focused tests, and documentation-impact receipts.

The job must not retrofit existing explainer specs or tracked HTML pages with
actual Mermaid diagrams. That content work belongs to a later Documentation
Curator pass.

execution-role record `validator-engineer@0.1.0--RT-20260611-001`.

## Claim Boundary

- Allowed: update visual-explainer and html-visual-explainer skill guidance,
  add the Mermaid Documentation subskill, add its validator examples and
  templates, add the local Mermaid runtime asset, integrate the structural
  validator into bootstrap validation, add focused tests, refresh the Markdown
  source registry, and record task-local control receipts.
- Forbidden: canonical ontology edits, manuscript TeX edits, PDF edits,
  tracked HTML content retrofits, generated wiki hand edits, generated HTML as
  independent authority, physics claim promotion, Gate Chair verdicts, public
  deployment, autonomous proof claims, or CDN Mermaid use in tracked HTML.
