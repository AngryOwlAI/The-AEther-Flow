# Director Decision Record DDR-20260612-004

## Current Objective

Create and execute one bounded `project-control-maintainer@0.1.0` task overlay
for summary-grounding contract hardening. The task removes prose source-list
grounding sentences from current tracked HTML explainer `summary_text` fields
and updates project-local explainer skills so future summaries keep source
grounding in `source_basis`.

## Accepted Design Decisions

- `summary_text` remains concise explanatory prose about the subject.
- Prose source-list grounding sentences are not valid summary prose for
  tracked explainers.
- `source_basis` remains the explicit grounding field for the summary block.
- Current source specs are updated first and generated HTML is refreshed from
  those specs.
- The project-local `html-visual-explainer` and `visual-explainer` skills carry
  the forward contract so newly generated summaries do not reintroduce the
  removed pattern.
