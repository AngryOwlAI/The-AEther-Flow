# Director Decision Record DDR-20260612-007

## Current Objective

Create and execute one bounded `project-control-maintainer@0.1.0` task overlay
to remove the obsolete `Analysis capsules` / `Claim-Aware Analysis` reader
layer and the repeated generic claim-boundary paragraph from the tracked HTML
explainer system.

## Role-Fit Matrix

| Candidate role | Fit | Reason |
| --- | --- | --- |
| `project-control-maintainer@0.1.0` | selected | The correction changes source-spec contracts, validator policy, role registration, helper tooling, tests, generated HTML, and task-control receipts. |
| `documentation-curator@0.7.0` | rejected | Appropriate for future explainer execution, but not for registering its own successor role or changing validator contracts. |
| `validator-engineer` | rejected | Validator guards are required, but they are supporting evidence rather than the full work surface. |
| physics roles | rejected | The task is documentation-system maintenance and must not promote scientific claims. |

## Accepted Design Decisions

- `analysis_capsule_schema` and `## Required Analysis Capsules` are no longer
  part of the active HTML explainer source-spec contract.
- `expandable_analysis_panels` is no longer a universal or valid current
  required control for tracked explainers.
- Section-level authority context remains allowed when it is specific and useful
  to the section, but the generic paragraph beginning `The legitimate claim is
  explanatory:` must not be appended to every content block.
- Generated HTML must not render the obsolete analysis section, analysis nav
  item, `data-analysis-capsule`, or `data-capsule-field` markers.
- Documentation Curator v0.7.0 supersedes v0.6.0 for future explainer work.

## Claim Boundary

- Allowed: update explainer source specs, active role guidance, project-local
  explainer skills, reader-layer helper tooling, validator guards, focused tests,
  generated tracked HTML derivatives, task-local control records, required
  registries, and generated memory/wiki surfaces.
- Forbidden: canonical ontology TeX edits, manuscript TeX edits, PDF edits,
  generated wiki hand edits, Mermaid diagram-definition changes, runtime Mermaid
  imports, generated HTML as independent authority, public deployment, physics
  claim promotion, Gate Chair verdicts, benchmark promotion, autonomous proof
  claims, and direct HTML-only edits without corresponding contract or
  source-spec authority.
