## Current Objective

Create and execute one bounded `validator-engineer@0.1.0` task overlay that
replaces browser-runtime Mermaid rendering in tracked HTML with build-time
inline SVG rendering. The job keeps the registered Markdown Mermaid source as
the canonical provenance surface, preserves the existing diagram shell and
zoom/pan interaction model, and makes standalone single-file HTML a validator

## Selected Role

The selected role is a one-job `validator-engineer@0.1.0` task overlay through
execution-role record `validator-engineer@0.1.0--RT-20260611-003`.

## Claim Boundary

- Allowed: update Mermaid documentation and HTML explainer skill contracts, add
  a subskill-local npm/Playwright renderer, tighten the Python Mermaid validator,
  regenerate the existing Mermaid-backed tracked HTML derivative with inline SVG,
  remove obsolete tracked Mermaid browser runtime assets after proof, add focused
  tests, refresh registries and generated memory/wiki derivatives, and record
  task-local control receipts.
- Forbidden: canonical ontology edits, manuscript TeX edits, PDF edits, physics
  claim promotion, Gate Chair verdicts, public deployment, generated HTML as
  independent authority, direct HTML-only content edits outside the registered
  source-backed regeneration proof, or reintroducing browser Mermaid runtime
  imports in tracked HTML.
