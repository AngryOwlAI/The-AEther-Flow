# Director Decision Record DDR-20260613-011

## Current Objective

Execute one bounded `project-control-maintainer@0.1.0` task overlay to create
the repo-local `user-modified-project` skill and integrate it into the
source-first registry and generated wiki system.

## Accepted Design Decisions

- The skill is repo-local at `.codex/skills/user-modified-project/`.
- The skill requires or recovers a brief human intent note before integration.
- The skill routes manual edits through existing classifiers and existing
  `continue-research` or `improve-project-system` workflows.
- Physics-bearing human edits remain proposals until research-control gates
  explicitly authorize stronger treatment.
- Generated wiki notes, generated registries, HTML, PDFs, and `.local/`
  retrieval surfaces remain derivatives and are refreshed only through the
  approved bootstrap path.

## Claim Boundary

- Allowed: create the repo-local skill contract, add UI metadata, update root
  AGENTS and README guidance, register the Markdown skill source, refresh
  generated registry/wiki/Obsidian surfaces, and create project-control
  receipts.
- Forbidden: canonical ontology TeX edits, manuscript TeX edits, PDF edits,
  generated wiki hand edits, generated HTML edits, physics claim promotion,
  Gate Chair verdicts, benchmark promotion, autonomous proof claims,
  checkpoint bypass, direct adoption of human physics edits, and generated
  derivatives as independent authority.
