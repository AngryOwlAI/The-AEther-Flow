Two Workflows

continue-research
  1. Resolve tracked handoff and active task state.
  2. Use Director control only when a bounded research job is needed.
  3. Execute at most one science AgentJob per invocation.
  4. Preserve claim boundaries and negative results.
improve-project-system
  1. Classify current changes and open project-system signals.
  2. Select one bounded non-scientific AgentJob.
  3. Update control, validator, memory, or documentation surfaces.
  4. Record documentation impact and run the gates.
